# 🌍 Eco-News Aggregator

## 📌 Project Description

The Eco-News Aggregator is a web application that collects and displays the latest environmental news from multiple sources. It helps users stay informed about sustainability, climate change, and other eco-related topics in real time.

## 🚀 Features

- Real-time News Fetching: Scrapes and updates environmental news from multiple sources.

- REST API Backend: Built with FastAPI to serve news data.

- PostgreSQL Database: Stores fetched articles to prevent duplicates.

- React Frontend: Displays news articles in a clean and minimal UI.

- Axios for API Calls: Fetches news from the backend.

## 🛠️ Technologies Used

### Frontend

- React.js

- Axios

### Backend

- FastAPI

- BeautifulSoup (for web scraping)

- SQLAlchemy (for database interactions)

- PostgreSQL (database)

- Uvicorn (for running FastAPI server)
